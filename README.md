# UberX x SpaceX - Space Rides

Maqueta estatica hecha solo con HTML5 y CSS3 para simular una app movil de viajes espaciales.

## Estructura

- `index.html`: pantalla de carga.
- `screens/`: pantallas del flujo principal.
- `css/app.css`: tokens visuales, componentes y responsive.
- `assets/img` y `assets/icons`: ilustraciones SVG optimizadas.

## Como abrir

Abre `index.html` en el navegador. No requiere instalacion, servidor, frameworks, librerias ni JavaScript.

## Interacciones CSS

- Navegacion entre pantallas con enlaces.
- Sheet expandible en `destination.html` usando `:checked`.
- Estados visuales con `:hover`, `:focus-visible` y transiciones.
- Layout responsive centrado en desktop y a ancho completo en mobile.
